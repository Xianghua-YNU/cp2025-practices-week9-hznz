# 复数动力学 - Mandelbrot集与Julia集实验报告

## 一、实验目的

1. 掌握复数迭代生成分形的基本原理
2. 理解逃逸时间算法(Escape Time Algorithm)的实现方法
3. 实现Mandelbrot集和Julia集的生成与可视化
4. 分析不同参数对分形图形的影响

## 二、实验过程

### 1. 核心算法实现

**逃逸时间算法实现:**
- 输入：复数参数c、最大迭代次数max_iter
- 处理：通过迭代公式z = z² + c计算逃逸时间
- 输出：每个点的逃逸时间(迭代次数)

**Mandelbrot集生成器:**
- 实现复数网格生成
- 记录每个c值的逃逸时间
- 返回2D逃逸时间数组

**Julia集生成器:**
- 实现复数网格生成
- 记录每个z0值的逃逸时间(固定c值)
- 返回2D逃逸时间数组

### 2. 分形实现

**Mandelbrot集参数:**
- 坐标范围：x ∈ [-2.0, 1.0], y ∈ [-1.5, 1.5]
- 最大迭代次数：100-200次
- 分辨率：800×800像素

**Julia集参数:**
- 坐标范围：x ∈ [-2.0, 2.0], y ∈ [-2.0, 2.0]
- 测试c值：-0.8+0.156j, -0.4+0.6j, 0.285+0.01j
- 最大迭代次数：100-200次
- 分辨率：800×800像素

### 3. 遇到的问题与解决
性能问题：
初始使用纯Python循环实现速度极慢
解决方案：改用NumPy向量化运算，性能提升约100倍

图像方向问题：
生成的图像与数学坐标系方向不一致
解决方案：使用.T转置数组并正确设置imshow的origin参数

颜色映射问题：
默认颜色映射对比度不足
解决方案：尝试多种colormap后选择'magma'，视觉效果最佳

## 四、分析与思考

1. **分形特性分析**
   - Mandelbrot集与Julia集的数学关系
Julia集的连通性取决于c值是否属于Mandelbrot集
当c在Mandelbrot集内时，对应的Julia集是连通的
三种Julia集案例中，前两种c值在Mandelbrot集内，故Julia集连通

   - 逃逸半径的选择对结果的影响
理论上任何大于2的值都适用
实际测试发现半径>2.5会丢失部分细节
半径<1.5会导致过早终止迭代
2. **参数影响实验**
   - 不同max_iter值的效果对比
迭代次数	计算时间(s)	图像质量
50	      0.8	       一般
100     	1.5	       良好
200	      3.0      	优秀
   - 分辨率与计算复杂度的关系
分辨率  	计算时间(s)	文件大小
400×400	  0.4	      200KB
800×800  	1.5	      800KB
1600×1600	6.0	      3.2MB
3. **扩展思考**
   - 如何实现平滑着色算法
```python
def smooth_color(iter, z):
    if iter == max_iter:
        return 0
    return iter + 1 - np.log(np.log(abs(z))/np.log(2))
```
   - 分形放大功能的实现思路
可通过交互式选择区域后重新生成网格
需要动态调整坐标范围和分辨率
   - 与L系统分形的比较
L系统更适用于规则分形(如科赫雪花)
复数迭代更适合复杂边界分形
计算复杂度相似，但可视化效果不同
4. **算法优化**
   - 向量化计算的优化效果
原始循环实现：约15秒
向量化实现：约0.15秒
提升约100倍
   - 并行计算的可能性
每个像素点计算独立
可使用multiprocessing或GPU加速
预计可再提升5-10倍性能
